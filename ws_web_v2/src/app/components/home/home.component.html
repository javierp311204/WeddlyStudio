PREGUNTAS FRECUENTES

Todo lo que necesitas saber

¿Es realmente gratis? ¿Qué incluye el plan Free? {{ openFaqIndex === 0 ? '−' : '+' }}
Sí, el plan Free es gratis para siempre. Incluye 1 boda activa, hasta 50 invitados, checklist básico y álbum de hasta 20 fotos. Sin tarjeta de crédito ni fecha de caducidad.
¿Pueden mis invitados confirmar asistencia (RSVP) sin crear una cuenta? {{ openFaqIndex === 1 ? '−' : '+' }}
Sí. Tus invitados reciben un enlace personalizado y confirman con un clic, sin registrarse. Solo el organizador necesita cuenta.
¿Qué diferencia hay entre Evento PRO y Premium? {{ openFaqIndex === 2 ? '−' : '+' }}
Evento PRO (49€) es un pago único para una boda, ideal si solo organizas la tuya. Premium (14€/mes) es para wedding planners o parejas con varias bodas activas simultáneas.
¿Puedo empezar gratis y pagar después cuando lo necesite? {{ openFaqIndex === 3 ? '−' : '+' }}
Exacto. Empieza con el plan Free, explora la plataforma y actualiza solo cuando necesites más capacidad. Todos tus datos se mantienen al cambiar de plan.
¿Son privados los datos de mi boda y mis invitados? {{ openFaqIndex === 4 ? '−' : '+' }}
Totalmente. Cada boda es un entorno aislado. Solo tú y las personas que invites expresamente tienen acceso. Cumplimos RGPD y nunca compartimos tus datos con terceros.
